# -
设计一个员工管理系统，用于记录和管理公司员工的基本信息，包括 姓名、工号、职位、薪资等 属性。 系统应提供员工信息的录入、查询、编辑和删除功能，并将员工信息保存到文件中。
该系统可以管理普通员工，经理和总裁三类。 并且可以利用文件交互功能来保存所添加的信息以及删除不需要的信息。
首先，先设计了基类Worker类，然后在利用基类派生了Employee（普通员工）类，Manager（经理）类和Boss（总裁）类。 接下来分别设计了各类的头文件和源文件，并分别在源文件中重新声名了在头文件中所写的函数。接下来设计了workerManager的头文件，在头文件中写了用户交互所需的各类功能的函数，并在源文件中分别单独设计和声名函数。最后利用system的源文件来让用户进行交互选择。
在Worker类的设计中利用纯虚函数·，保证后续通过基类指针来指向不同的派生类对象，实现多态性。
在编号排序函数实现时利用了冒泡排序的算法来实现。
